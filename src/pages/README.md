# ArcByte Command Center

## Project Overview

ArcByte Command Center is a modern AI-powered SaaS dashboard built using React, TypeScript, and Tailwind CSS. It provides a centralized platform for managing projects, AI agents, analytics, team members, and application settings.

## Features

* Dashboard Overview
* Project Management Module
* AI Agent Center
* Analytics Dashboard
* Team Members Page
* Settings Module
* Login Page
* API Integration with DummyJSON
* Loading States
* Error Handling
* Responsive Design
* Reusable Components
* React Router Navigation

## Tech Stack

* React.js
* TypeScript
* Tailwind CSS
* React Router DOM
* Axios
* TanStack React Query
* Recharts
* Vite
